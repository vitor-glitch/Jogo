O projeto Space Blaster foi desenvolvido com o objetivo de aplicar conceitos de programação e lógica computacional na criação de um jogo interativo. A proposta central consiste em uma aventura espacial, na qual o jogador controla uma nave e precisa enfrentar desafios, desviar de obstáculos e alcançar a maior pontuação possível.
Durante o desenvolvimento, foram utilizados conhecimentos em linguagem de programação (como JavaScript), além de conceitos importantes como manipulação de eventos, controle de animações e lógica de colisão. O jogo foi estruturado para ser simples, mas ao mesmo tempo desafiador, proporcionando uma experiência dinâmica ao usuário.
Um dos principais focos do projeto foi a criação de um sistema de pontuação e ranking, incentivando a competitividade entre os jogadores. Além disso, foram implementados elementos visuais e sonoros para tornar a jogabilidade mais envolvente.
Também foi usado a IA para me direcionar sobre oq fazer e como fazer o código.
Ao longo do processo, foram enfrentados desafios como a organização do código, a correção de bugs e a implementação de novas funcionalidades, o que contribuiu significativamente para o aprimoramento das habilidades em desenvolvimento de software.
O Space Blaster representa não apenas um jogo, mas também uma evolução no aprendizado prático de programação, demonstrando criatividade, resolução de problemas e aplicação de conceitos técnicos em um projeto real.
